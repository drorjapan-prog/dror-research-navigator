# dror-research-navigator — 働く人の心理学・組織研究ナビ

## 正規サイト

**https://research.dror.co.jp/**

`site/` がWeb配信ルートです（Cloudflare Pages / Build output directory = `site`）。サイトとしての詳細な説明は [site/README.md](site/README.md) を参照してください。

## このプロジェクトの目的

働く人の心理学・組織研究 25分野について、研究の主張・出典・限界・方法・用語を、広告・追跡なしで読み、確かめ、引用・再利用できる形で公開する、独立した公共目的の研究案内プロジェクトです（運営: DroR Corporation）。

本ナビと原著・訂正・撤回・現行の公式資料の記述が異なる場合は、元の情報源を優先してください。

## データへのアクセス

- オープンデータの方針と一覧: https://research.dror.co.jp/open
- 分野別ダウンロード（ZIP 25本・オフライン閲覧可）: https://research.dror.co.jp/downloads/
- 各分野の `data/` 配下に claims / studies / glossary / protocol / screening / quality（JSON・CSV）、`references.bib`・`references.ris`、JSON Schema、Dataset JSON-LD を収載
- 調査・更新方針: https://research.dror.co.jp/methodology

## 引用方法

- このリポジトリ・データセット全体の引用: ルートの [CITATION.cff](CITATION.cff)（GitHubの「Cite this repository」からも取得できます）
- 分野・ページ単位の引用形式（例・第1分野）: https://research.dror.co.jp/psychological-safety/pages/citation-formats
